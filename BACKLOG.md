# Backlog

Feature ideas collected during development.
Do not implement these until the current build step is complete.

## Ideas

- Custom exercise creation (name, image, muscle group)
- OAuth login (Google, email magic link)
- Export session history to CSV or PDF (currently JSON only)
- Dark mode
- Social sharing ("I just hit a new PR!")
- Video demos for each exercise
- Superset support (two exercises with no rest between)
- RPE (Rate of Perceived Exertion) tracking per set
- Push notifications via Web Push API (rest timer done, daily workout reminder)
- Coach / team accounts (one coach manages multiple athletes)
- Annual plan gifting (buy premium for a friend)
